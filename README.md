# 🎬 BookMyMovie - Java JDBC Project

## 📌 Overview
Backend console-based movie ticket booking system built using Java, JDBC, and MySQL.

## 🚀 Features
- View theatres by city
- View shows
- Book seats
- Prevent double booking

## 🛠 Tech Stack
- Java
- JDBC
- MySQL

## ▶️ How to Run
1. Setup MySQL DB: `bookmymovie`
2. Update DB credentials in `DataBaseConfig.java`
3. Run `BookMyMovieApp.java`

## 📂 Project Structure
- config/
- entity/
- service/
